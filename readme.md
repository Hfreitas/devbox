<h1 align="center">Devbox</h1>

Easy and fast way to get development environments using Docker

## Services

#### Mysql 5.7

```bash
make mysql5.7-up
make mysql5.7-down
make mysql5.7-status
make mysql5.7-logs
make mysql5.7-info
```
