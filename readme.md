# Devbox

Easy way to up development environments using Docker

## Services

#### Mysql 5.7

```bash
make mysql5.7-up
make mysql5.7-down
make mysql5.7-status
make mysql5.7-logs
make mysql5.7-info
```

#### Postgres 13.2

```bash
make postgres13.2-up
make postgres13.2-down
make postgres13.2-status
make postgres13.2-logs
make postgres13.2-info
```
